# shifter-tts 🎙️

**Десктопное приложение для синтеза речи (TTS) на основе Qwen3-TTS с поддержкой voice cloning, калибровки голоса и транскрипции.**

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ Features / Возможности

| Функция | Описание |
|---------|----------|
| 🎭 **Голосовые инструкции** | `[смеётся]`, `[шёпотом]`, `[громко]` и другие прямо в тексте |
| 🗣️ **Voice Cloning** | Клонирование голоса по референс-аудио |
| 🎛️ **Калибровка голоса** | Смешивание нескольких векторов для точного контроля |
| 📝 **Транскрипция** | Audio → текст (Whisper) на 6 языках |
| 🖥️ **GPU мониторинг** | VRAM, температура и загрузка в реальном времени |
| 🎬 **Видео-аудио** | Извлечение аудио из видеофайлов |

---

## 📋 Requirements / Требования

- **Python ≥ 3.10**
- **GPU NVIDIA с ≥ 8 ГБ VRAM** (для inference)
- **GPU NVIDIA с ≥ 12 ГБ VRAM** (для дообучения / finetuning)
- Linux / Windows / macOS

## 🚀 Quick Start (Быстрый старт) / Установка

**Просто запустите приложение — всё установится само:**

### Windows
```batch
# Скачайте репозиторий и запустите:
git clone https://github.com/50shifter/shifter-tts.git
cd shifter-tts
run.bat
```

### Linux / macOS  
```bash
# Скачайте репозиторий и запустите:
git clone https://github.com/50shifter/shifter-tts.git  
cd shifter-tts
chmod +x run.sh
./run.sh
```

**Что происходит автоматически:**
1. Проверка Python (нужен 3.10+)
2. Создание виртуального окружения (venv), если нет
3. Установка всех зависимостей (pip install)
4. Скачивание модели Qwen3-TTS (~4 ГБ)
5. Запуск GUI приложения

> ⚠️ **Если модель требует принятия лицензии на HuggingFace:**
> 1. Зарегистрируйтесь бесплатно: https://huggingface.co/join
> 2. Примите условия на странице модели
> 3. Запустите `huggingface-cli login`
> 4. Повторно запустите `run.bat` / `run.sh`

### Ручная установка (минимальные шаги)
```bash
# 1. Виртуальное окружение
python -m venv venv
source venv/bin/activate        # Linux/macOS
venv\Scripts\activate           # Windows

# 2. Установка зависимостей
pip install -r requirements.txt

# 3. Запуск приложения
python -m tts_app
```

## 📥 Download Model (Загрузка модели)

### Как это работает:

1. **При первом запуске** модель скачивается автоматически из HuggingFace Hub (~4 ГБ)
2. **Бесплатно и без ключа** — все модели публичные, скачиваются сразу
3. Модель сохраняется в `~/.cache/huggingface/hub/` и используется офлайн

### Ручная загрузка:
```bash
# Базовая модель (voice cloning)
python download_model.py base

# Кастомная версия с голосом
python download_model.py custom

# Voice Design (генерация голоса по описанию)
python download_model.py voice_design

# Только токенизатор (для обучения)
python download_model.py tokenizer
```

## 📖 Usage / Использование

### Голосовые инструкции в тексте

Вставляйте инструкции в квадратных скобках прямо в текст:

| Инструкция | Описание | Пример звучания |
|------------|----------|-----------------|
| `[смеётся]` | Смех | весёлый, с хохотом |
| `[хохочет]` | Громкий смех | раскатистый хохот |
| `[грустно]` | Грустный тон | тихий, подавленный |
| `[радостно]` | Радостный тон | энергичный, весёлый |
| `[сердито]` | Сердитый тон | резкий, раздражённый |
| `[шёпотом]` | Шёпот | тихий шёпот |
| `[громко]` | Громко | усиленная громкость |
| `[пауза]` | Пауза ~0.5 сек | тишина |
| `[длинная_пауза]` | Пауза ~1.0 сек | длинная тишина |

**Примеры:**
```
Привет [смеётся] как дела?
[шёпотом] представь себе тихую комнату с дождём за окном
Знаешь что? [шепч] [смеётся] я сегодня выиграл! [пауза] Серьёзно.
```

### Калибровка голоса

Добавьте несколько `.pt` векторов голоса с разными эмоциями:
1. Добавьте файлы .pt (грустный, весёлый, шёпот)  
2. Настройте вес каждого (1.0 = полный, 0.5 = половина)
3. Нажмите «Калибровать» → получите смешанный вектор
4. Сохраните как новый .pt и используйте в TTS

### Транскрипция

**Поддерживаемые форматы:** WAV, MP3, OGG, FLAC, M4A  
**Языки:** English, Русский, German, French, Spanish

## ❓ FAQ

### Нужно ли регистрироваться на HuggingFace?
**Нет, если модель публичная.** Большинство моделей скачиваются сразу без регистрации.

Если модель требует принятия лицензии (gated model):
1. Бесплатная регистрация: https://huggingface.co/join
2. Примите условия на странице модели
3. Авторизация через терминал: `huggingface-cli login`
4. Модель начнёт скачиваться автоматически

### Почему не скачивается модель?
- Проверьте интернет-соединение
- Убедитесь что вы приняли лицензию (если требуется)
- Попробуйте: `python download_model.py base`

### Можно использовать без GPU?
Да, но **очень медленно** (секунды на одно предложение). Для нормальной работы нужна NVIDIA GPU с ≥8 ГБ VRAM.

### Где хранится модель после скачивания?
`~/.cache/huggingface/hub/` — используется офлайн, повторно не скачивается.

## ⚠️ Notes / Примечания

1. **Первый запуск** — модель скачается автоматически (~4 ГБ)
2. **GPU обязательно** — без GPU работа будет крайне медленной (только CPU inference)
3. **Voice cloning** — качество зависит от длины и чистоты референс-аудио
4. Инструкции в `[скобках]` влияют только на следующий фрагмент текста
