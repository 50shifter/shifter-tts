# shifter-tts

Десктопное приложение для синтеза речи на основе Qwen3-TTS. Поддержка клонирования голоса, калибровки и транскрипции.

## Возможности

- Голосовые инструкции прямо в тексте: `[смеётся]`, `[шёпотом]`, `[громко]`
- Voice Cloning — клонирование голоса по короткому аудио
- Калибровка — смешивание нескольких голосовых векторов для точного контроля
- Транскрипция — Audio → текст (Whisper, 6 языков)
- Мониторинг GPU — VRAM, температура, загрузка в реальном времени
- Аудио из видео — извлечение звука из mp4, mov и других форматов

## Требования

- Python 3.10+
- NVIDIA GPU с 8 ГБ VRAM (для синтеза)
- NVIDIA GPU с 12 ГБ VRAM (для обучения модели)
- Windows, Linux или macOS

## Установка

Просто запустите — всё установится автоматически:

### Windows
```batch
run.bat
```

### Linux / macOS
```bash
./run.sh
```

**Что происходит:**

1. Виртуальное окружение (venv) создаётся автоматически
2. Зависимости устанавливаются через pip
3. Модель скачивается (~4 ГБ)
4. Приложение запускается

> **Если модель требует принятия лицензии HuggingFace:**
> 1. Зарегистрируйтесь: https://huggingface.co/join
> 2. Примите условия на странице модели
> 3. В терминале: `huggingface-cli login`
> 4. Запустите снова

### Ручная установка

```bash
python -m venv venv
source venv/bin/activate   # Linux/macOS
venv\Scripts\activate      # Windows

pip install -r requirements.txt
python -m tts_app
```

## Использование

### Голосовые инструкции

Вставляйте указания в квадратных скобках прямо в текст:

- `[смеётся]` / `[хохочет]`
- `[грустно]` / `[радостно]`
- `[шёпотом]` / `[громко]`
- `[пауза]` / `[длинная_пауза]`

Пример:
```
Привет [смеётся] как дела?
```

### Клонирование голоса

1. Загрузите короткое референсное аудио (10-20 секунд)
2. Введите текст для синтеза
3. Нажмите «Synthesize»

### Калибровка голосовых атрибутов

1. Загрузите несколько `.pt` файлов с разными голосовыми атрибутами
2. Настройте вес каждого
3. Нажмите «Calibrate» → появится новый микс
4. Сохраните результат как новый `.pt` и используйте при синтезе

### Транскрипция

- Форматы: WAV, MP3, OGG, FLAC, M4A
- Языки: English, Русский, German, French, Spanish

## Вопросы

### Нужен ли ключ HuggingFace?

Нет, если модель публичная. Модели скачиваются автоматически.
Если требует принятия лицензии — зарегистрируйтесь и войдите через терминал: `huggingface-cli login`

### Модель не скачивается?

- Проверьте интернет
- Убедитесь, что вы приняли условия лицензии на странице модели
- Попробуйте: `python download_model.py base`

### Можно без GPU?

Да, но медленно. Адекватная работа потребует NVIDIA GPU с 8 ГБ VRAM.

### Где хранится модель?

`~/.cache/huggingface/hub/` — не скачивается повторно.
