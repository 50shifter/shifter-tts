# 📖 Памятка пользователя shifter-tts

## 🎭 Голосовые инструкции в скобках

Вставляй инструкции в квадратных скобках прямо в текст. Модель воспринимает их
как указания по интонации, громкости и эмоциям.

### Эмоции
| Инструкция    | Описание              | Как звучит               |
|---------------|-----------------------|--------------------------|
| `[смеётся]`   | Смех                  | весёлый, с хохотом        |
| `[хохочет]`   | Громкий смех          | раскатистый хохот         |
| `[грустно]`   | Грустный тон          | тихий, подавленный       |
| `[радостно]`  | Радостный тон         | энергичный, весёлый      |
| `[сердито]`   | Сердитый тон          | резкий, раздражённый     |
| `[удивлённо]` | Удивлённый тон        | возвышенный, удивлённый  |
| `[плачет]`    | Плач                  | дрожащий, жалобный       |
| `[вздыхает]`  | Вздох                 | выдох, усталость         |

### Громкость
| Инструкция    | Описание              |
|---------------|-----------------------|
| `[шёпотом]`   | Шёпот                 |
| `[тихо]`      | Тихо                  |
| `[громко]`    | Громко                |
| `[кричит]`    | Крик                  |

### Паузы
| Инструкция       | Описание           |
|------------------|--------------------|
| `[пауза]`        | Пауза ~0.5 сек     |
| `[длинная_пауза]`| Пауза ~1.0 сек     |

### Английские инструкции
Работают те же команды на английском:
- `[laughing]`, `[whispering]`, `[shouting]`, `[pause]`, `[long_pause]`

## 💡 Примеры

### Одна инструкция
```
Привет [смеётся] как дела?
```

### Несколько инструкций
```
Знаешь что? [шепч] [смеётся] я сегодня выиграл! [пауза] Серьёзно.
```

### С ударениями
Инструкции НЕ заменяют ударения — пиши и то, и другое:
```
При́вет [грустно] как же де́ла?
```

### Несколько инструкций подряд
Можно ставить несколько подряд — они применяются к следующему фрагменту текста:
```
[шёпотом] представь себе [грустно] тихую комнату с дождём за окном.
```

## ⚠️ Важное

1. Инструкция влияет **только на следующий за ней фрагмент** текста
2. Следи за **ударениями** — модель их не понимает автоматически
3. Инструкция **не произносится** — она скрытый токен для модели
4. Если инструкция неизвестна — модель просто пропустит её с предупреждением

## 🔧 Калибровка голоса

Добавляй несколько `.pt` векторов голоса с разными эмоциями:
- Грустный, весёлый, шёпот — для смешанного результата
- Настрой вес каждого (1.0 = полный, 0.5 = половина)
- Нажми «Калибровать» → получишь смешанный вектор
- Сохрани как `.pt` и используй в TTS

## 🎙️ Транскрипция

Поддерживаемые форматы: WAV, MP3, OGG, FLAC, M4A
Языки: English, Русский, German, French, Spanish

## 🖥️ GPU

Мониторинг VRAM, температуры и использования GPU в реальном времени.
Требование для дообучения: GPU с >= 12 ГБ VRAM
